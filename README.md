A simple console-based Snake and Ladder game implemented in Java, designed with a focus on clean object-oriented principles and low-level design. The game supports multiple players, different board configurations (standard, random with difficulty levels, and custom setups), and uses design patterns like Strategy, Factory, and Observer to keep the code modular and extensible. Players take turns rolling a dice, moving across the board, and interacting with snakes and ladders until one reaches the final cell and wins.
